# Apple Trajectory Prediction

Модуль предсказания траектории яблока по данным от детектора объектов.

## Структура проекта

 Файл                   Описание                                      

 main.py                Запуск, симуляция, сборка всего модуля        
 tracker.py             Буфер последних точек траектории              
 physics_predictor.py   Физическая модель прогноза (g, vx, vy)        
 evaluator.py           Расчет ошибок и статистики                    
 visualizer.py          Графики траектории и ошибок                   
 config.py              Настройки проекта                             

## Установка

pip install -r requirements.txt

## Запуск

python main.py

## Входные данные

Модуль ожидает от детектора на каждый кадр:
    (timestamp, x, y, confidence)

## Выходные данные

- Предсказанная позиция яблока через PREDICT_DELTA секунд
- Ошибка предсказания в пикселях
- Статистика по серии предсказаний (MAE, std, max)
- Графики траектории и ошибок
